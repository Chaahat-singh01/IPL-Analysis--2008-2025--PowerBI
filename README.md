# IPL Analysis 2025 Dashboard

## Project Overview

This Power BI dashboard provides insights into IPL from 2008 to 2025 performance using match and ball-by-ball data.

## Key KPIs

* Total Matches
* Total Runs
* Total Wickets
* Strike Rate
* Economy Rate

## Features

* Orange Cap Analysis
* Purple Cap Analysis
* Team Performance Comparison
* Venue Analysis
* Season-wise Insights
* Interactive Filters and Slicers

## Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling
